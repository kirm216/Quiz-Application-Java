This is a console-based Quiz Application built using Java and Object-Oriented Programming concepts. The program asks multiple-choice questions and shows the final score at the end.

Features:
Multiple choice questions
Score calculation
Uses OOP concepts like classes and objects
Simple menu-based flow
Clean Java code

Technologies Used:
Java
OOP concepts
Arrays and loops

Project Structure:
Question.java - Contains the Question class
Quiz.java - Contains the Quiz engine
QuizApp.java - Main file that runs the quiz

How It Works:

The program displays each question with options.

The user enters an answer.

The program checks if the answer is correct.

The score is updated.

At the end, the final score is shown.

How To Run:
Step 1: Compile the files using:
javac Question.java Quiz.java QuizApp.java
Step 2: Run the program using:
java QuizApp
